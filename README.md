# Netflix Data Analysis

# Project Overview
This project describes about the data analysis of Netflix Movies and TV shows Dataset.
This is mainly used to practice the data anlysis, data cleaning, and visualization using the tools and libraries.
It aims to know how the raw data can be cleaned and visualize the requirement accordingly.


# About Dataset
The Dataset taken from the kaggle and the name of the data set is (Netflix movies and TV shows)
Taken as a raw data and cleaned
Used the pandas library to clean the raw data
